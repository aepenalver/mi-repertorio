# Mi Repeertorio

Desafío 2 para mostrar conocimientos de desarrollo con Node.Js.

## Tecnologías Empleadas

- JavaScript
- Node JS

## Requisitos

- Tener instalado [Node.js](https://nodejs.org/) (versión 12 o superior).
- Una terminal o consola de comandos.

## Instalación

Instrucciones para instalar y usar la aplicación:

1. Clonar repositorio:

```bash
git clone https://github.com/aepenalver/mi-repertorio.git

```

2. Acceder al directorio:

```bash
cd mi-repertorio
```

3. Configuración
   Abrir terminal o consola de comandos y ejecutar

```
npm install
npm run dev
```

## Uso

Abrir navegador web y dirigirse a http://localhost:3000
